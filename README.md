How to Run it?

  Use Grunt and Bower
  You must already install node.js.
  go to the swan root
  $npm install --global
  $npm install --global  bower grunt-cli compass
  $npm install
  $bower install
  Start the server: $grunt server
  Build the file to dist: $grunt build
