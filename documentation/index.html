Introduction

"Swan" is a fully responsive admin web App built with AngularJS. It use Sass CSS which makes it easy to modify.

Features

  • Responsive Design.
  • Web App (Single Page Application) Built with AngularJS. Fully AJAX powered, never refresh the entire page again. Faster, smoother.
  • Runnable Task Application. Try it out.
  • Solid Workflow with Yeoman, scaffolding with Yo, dependencies (plugins) managment with Twitter Bower, tasks automation with Grunt, make all things organised.
  • Realtime Form Validation.
  • Build with Sass CSS (Scss files are included). Easy modification with variables.
  • Many widgets and components.
  • Light Weight.
  • Built with Bootstrap 3.
  • Valid HTML5 and CSS3
  • Cross Browsers
  • And much more.

Automation with Grunt

Grunt is a JavaScript task runner (Automation), save yourself from repetitive tasks.

"Gruntfile.js"
is the configure file of Grunt (You can click the link to know more). It is used to configure or define tasks and load Grunt plugins.
"package.json"
is the JSON file of Grunt (You can click the link to know more). This file is used by npm to store metadata for projects published as npm modules. You will list grunt and the Grunt plugins your project needs as devDependencies in this file.

Inside package.json file, you'll see all the Grunt plugins that help to automate the tasks.

And all the plugins are configured in Gruntfile.js file.

How to Use It (Develop with It)?

In order to do development with it, you'll need have Node.js (NPM comes along with it), Yeoman (Bower, Grunt come along with it), Ruby, Sass , Compass installed.

And you'll need to know how to use them, click the links to learn more if you not familiar with these yet.

If you already know how to use them, then just do a few things to start developing:

  1. Enter "Swan" folder with your command line.
  2. Install NPM and Bower packages with NPM and Bower.
  3. Then type grunt server and you are ready to go.

All the files for development are under ./swan/app/ folder.

Structures of the folders inside app folder.

app/
├── fonts/
├── images/
├── scripts/
├── styles/
├── views/

contains all the custom javascripts.

contains all the template files (HTML).

the favicon icon.
index.html
The main HTML file.

Basic Template Structure

The basic structure of template (index.html) is quiet simple. It contains 3 parts: Head, which load all the stylesheets. Body, which contains the content App. Foot , which load all the scripts.
