Angular Map

Simple to use directive for easy embedding and interacting with a map managed with the leaflet map library on an AngularJS application.

Open Street Map

This directive allows you to embed a map on your AngularJS application and interact bi-directionally with it via the AngularJS scope and the leaflet map library API. In its current state, it's not a full-featured library, although it's easy to use and extend, and a good starting point to learn to code a directive into the great AngularJS library.

The code of the directive and this page has been inspired by the Angular Google Maps directive developed by nlaplante. The initial code has been (and is being) evolved and improved by these contributors.
